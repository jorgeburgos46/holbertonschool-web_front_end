# holbertonschool-web_front_end

Web front-end projects from Holberton School / Holberton coursework.

## html_advanced

Project focused on HTML5: semantic tags, document structure, media
integration, tables, lists, and W3C-compliant markup. No CSS is used in
this project on purpose — the goal is to master structure, not styling.

Company used across the exercises: **Techium**.
